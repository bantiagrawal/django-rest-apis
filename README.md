# blockchainmetrics

# Prerequisit
- develop and tested with Python 3.9.2
- pip command should work
# Installation:
blockchainmetrics app for Prahlad Vadap
- unzip file 
- go inside extracted folder
- run 'pip install -r requirements.txt'
- run 'python bcmetrics/manage.py runserver'
